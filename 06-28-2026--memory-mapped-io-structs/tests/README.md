# tests/

The behavioral self-test that grades this exercise lives in the simulation
harness and is invoked by the build commands in `BUILD.md`. You do not need to
add tests here to pass.

If you want to experiment, you may write your own throwaway probe in a separate
file and compile it yourself — but **do not** modify the grading flow, and the
only files that count toward grading are `include/qtimer.h` and `src/qtimer.c`.
